# Healthcare Business Registration Portal

Digital registration platform for traditional healthcare businesses and wellness startups.

## About

This portal facilitates the registration and certification of healthcare businesses specializing in traditional medicine and wellness services. The platform provides comprehensive services including streamlined registration, document verification, investment opportunities, and business networking.

## Features

- **Efficient Registration**: Streamlined business registration workflow
- **Document Verification**: Secure document upload and admin verification
- **Investment Platform**: Connect businesses with potential investors
- **Multi-User System**: Separate portals for businesses, admins, and investors
- **Secure Authentication**: Multi-layer security with encrypted data storage
- **Responsive Design**: Optimized for all devices

## Technologies Used

- **React 18** - Modern UI library
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first CSS framework
- **Supabase** - Backend-as-a-Service for authentication and database
- **React Router** - Client-side routing
- **Vite** - Fast build tool and development server

## Development

To run the project locally:

```bash
# Clone the repository
git clone [repository-url]
cd ayush-startup-portal

# Install dependencies
npm install

# Start development server
npm run dev
```
